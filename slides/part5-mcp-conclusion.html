Rappel : MCP en 2 minutes

MCP (Model Context Protocol) est un standard ouvert qui permet à Claude Code de se connecter à des outils et sources de données externes via un protocole unifié.

Les 3 modes de transport
stdio

~5ms — Serveurs locaux (90% des cas)

SSE

~45ms — Serveurs distants / équipe

HTTP streamable

Nouveau standard (remplace SSE)

Commandes essentielles 
claude mcp add github -- npx -y @modelcontextprotocol/server-github

claude mcp list

/mcp

Pour tout le détail : consultez le repo du workshop MCP précédent.

Ce que MCP permet concrètement

💡 Analogie : MCP est le "USB-C" de l'IA — un seul standard pour brancher n'importe quel outil.

🐙
GitHub + Jira 
Implémente la feature décrite dans le ticket JIRA ENG-4521 et ouvre une PR GitHub
📊
Sentry + monitoring 
Analyse les erreurs Sentry de la semaine et résume les problèmes critiques
🗄️
PostgreSQL 
Trouve dans notre base les 10 users les plus actifs ce mois
